Principais formatações

Negrito / Destaque

Nesta frase tenho um termo em destaque usando a tag <strong/> pois é semântica

Itálico / Ênfase

Nesta frase tenho um termo em ênfase usando a tag <em/> pois é semântica

Texto marcado

Podemos criar também um texto marcado usando a tag <mark/>

Mudamos a cor do texto marcado usando a tag CSS3 <style/> dentro da tag <mark/>

Texto grande / pequeno

Para criar um texto pequeno podemos usar a tag <small/>
